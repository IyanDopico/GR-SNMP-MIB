# GR-SNMP-MIB
Implementación de un sistema de gestión de red mediante SNMP para la monitorización de un servidor FTP monohilo. Este repositorio incluye el diseño de un MIB personalizado (OID 1.3.6.1.4.1.47454) validado con severidad 6, el desarrollo del agente utilizando SNMP Agent Builder y la configuración de métricas, gráficas y alertas en Zabbix.
